# StayHealthy - Instant Doctor Appointment & Consultation App

StayHealthy သည် လူနာများအတွက် ဆရာဝန်များနှင့် အချိန်မရွေး၊ နေရာမရွေး ရက်ချိန်းယူနိုင်ရန်နှင့် ကျန်းမာရေးဆိုင်ရာ အကြံဉာဏ်များ ရယူနိုင်ရန် တည်ဆောက်ထားသော ခေတ်မီ Web Application တစ်ခု ဖြစ်သည်။

## 🌟 Key Features
* **User Authentication:** လုံခြုံစိတ်ချရသော Sign-Up နှင့် Login စနစ် (JWT Token အသုံးပြုထားသည်)။
* **Instant Consultation:** ဆရာဝန်များနှင့် ချက်ချင်း ဆက်သွယ်နိုင်သော စနစ်။
* **Doctor Search & Filter:** အထူးကုဘာသာရပ်အလိုက် ဆရာဝန်များကို ရှာဖွေနိုင်ခြင်း။
* **Appointment Management:** ရက်ချိန်းယူခြင်း၊ ပြန်လည်စစ်ဆေးခြင်းနှင့် ဖျက်သိမ်းခြင်း (Cancel) လုပ်ဆောင်ချက်များ။
* **Patient Reports:** ဆေးစစ်ချက်မှတ်တမ်းများကို PDF အနေဖြင့် ကြည့်ရှုနိုင်ခြင်း။
* **Responsive Design:** ဖုန်း၊ Tablet နှင့် Desktop အားလုံးတွင် အဆင်ပြေပြေ အသုံးပြုနိုင်ခြင်း။

## 🛠️ Tech Stack
* **Frontend:** React.js (Hooks, Functional Components)
* **Routing:** React Router DOM v6
* **State Management:** React Context API / Session Storage
* **Backend Support:** Express.js (OnRender Hosted API)
* **Design Tool:** Figma (UI/UX Prototyping)
* **Styling:** CSS3, Bootstrap 5

## 🚀 Getting Started

### Prerequisites
* [Node.js](https://nodejs.org/) (v16 or higher)
* npm (Node Package Manager)

### Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/sithu199/med_appt.git
    cd med_appt
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the application:**
    ```bash
    npm start
    ```
    Application သည် `http://localhost:3000` တွင် စတင်ပတ်မောင်းနေပါလိမ့်မည်။

## 📂 Project Structure
* `src/components/` - UI components များ (Navbar, Login, SignUp, etc.)
* `src/pages/` - Application ၏ main pages များ
* `public/` - Static assets များနှင့် index.html
